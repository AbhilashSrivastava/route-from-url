# Route From URL

Creates Route Object from URL.



### Usage

Use `getRoute` function which takes in url(string) and returns the route object.

Format for route object:
```
   getRoute(url: string)
      => {
            href: string,
            protocol: string,
            host: string,
            hostname: string,
            port: string,
            pathname: string,
            search: string,
            hash: string
         }

```
